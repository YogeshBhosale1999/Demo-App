# Demo App

A simple demo application to showcase Git workflows, branching strategies, and CI/CD integration.

## 🚀 Features
- Basic application setup
- Demonstrates Git branching (`main`, `staging`, `feature`)
- Pull request workflow for collaborative development
- Ready for CI/CD pipeline integration

## 📂 Branching Strategy
- **main** → Stable production branch
- **staging** → Pre-production branch for QA testing
- **feature** → Developer branch for new features
